# Sudoku App
A personal ad-free Sudoku web app built with React.

## Goal
I am building this project to improve my programming skills and create a Sudoku app I enjoy using.

## Tech stack
- React
- JavaScript
- CSS
- Vite

## Planned features
- 9x9 Sudoku board
- Number input
- Mistake checking
- Timer
- Notes mode
- Multiple difficulty levels
