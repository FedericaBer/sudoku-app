# Sudoku App

A personal ad-free Sudoku web app built with React.

## Live Demo

Play Sudoku: https://sudoku-app-blush.vercel.app/

## Tech stack

- React
- JavaScript
- CSS
- Vite

## Planned features

- 9x9 Sudoku board
- Number input
- Mistake checking
- Timer
- Notes mode
- Multiple difficulty levels
